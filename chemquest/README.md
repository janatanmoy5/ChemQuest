# 🧪 ChemQuest

ChemQuest is a Streamlit web application for exploring chemical compounds.  
It fetches chemical information from **PubChem** and **ChEMBL**, retrieves PubMed references, and summarizes abstracts.  

---

## Features

- **Chemical Identity:** Molecular formula, weight, PubChem CID, ChEMBL ID  
- **Chemical Properties:** XLogP, H-bond donors/acceptors, TPSA, rotatable bonds  
- **Chemical Classification:** Infers type and mechanism of action  
- **Literature:** Fetches recent PubMed references  
- **AI Summary:** Lightweight summarization of abstracts
